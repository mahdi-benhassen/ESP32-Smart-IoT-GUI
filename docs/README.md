## Quick Links

- **Web Interface**: `http://{gateway-ip}/`
- **API Base**: `http://{gateway-ip}/api/v1`
- **Default Credentials**: `admin` / `admin123`

## Features

- WiFi Configuration (Station + AP mode)
- Bluetooth LE Gateway
- ZigBee Network (ESP32-H2)
- MQTT Broker Integration
- Node Management
- Real-time Sensor Monitoring
- REST API with Events
